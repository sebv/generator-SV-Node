# Generator-SV-Node

My own node/mocha generator for Yeoman.

## Getting started
- Make sure you have [yo](https://github.com/yeoman/yo) installed:
    `npm install -g yo`
- Install the generator: `npm install -g generator-SV-Node`
- Run: `yo SV-Node`
